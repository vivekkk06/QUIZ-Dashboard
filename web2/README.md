# Interactive Quiz Application

## Description
A dynamic, client-side web application that allows users to test their knowledge through a timed, multiple-choice quiz. The application evaluates answers in real-time, manages strict time limits per question, and provides a final performance summary.

## Features Implemented
* **Dynamic Rendering:** Questions and options are injected into the DOM via JavaScript, allowing for easy expansion of the question bank.
* **Per-Question Timer:** A countdown timer runs for each question (15 seconds). If the timer reaches zero, the correct answer is revealed, and the quiz automatically advances.
* **Instant Feedback:** Answers are immediately validated, highlighting correct answers in green and incorrect ones in red.
* **Score Tracking:** Accurate evaluation of the user's score based on correct selections made within the time limit.
* **Responsive UI:** Clean, modern, and accessible styling using CSS flexbox and CSS variables.

## How to Run
1. Ensure all three files (`index.html`, `style.css`, and `script.js`) are saved in the same directory.
2. Open the `index.html` file directly in any modern web browser (Chrome, Firefox, Safari, Edge).
3. No local server or build tools are required.
